Überschriften-Struktur

Bei Überschriften gibt es einen Unterschied zwischen nativen Anwendungen und hybriden Anwendungen mit eingebettetem HTML-Inhalt. Während es in HTML sechs Überschriftsebenen h1 bis h6 gibt, gibt es bei nativen Apps keine Überschriftenebenen.

Ausgabe von TalkBack

Native Apps

Sie haben vielleicht schon bemerkt, dass z. B. die Überschrift "Alternativtexte", im Gegensatz zu der Überschrift "Überschriften-Struktur" nur als "Überschrift Alternativtexte" angekündigt wurde. Dies ist ein Text, der auf nicht-visueller Ebene als Überschrift formatiert wurde.

HTML Inhalt

Bei HTML-Dokumenten sagen mobile Screenreaders zum Beispiel "HTML-Inhalt Überschrift 3". Da es sich um eine Überschrift der dritten Überschriften-Ebene (h3) handelt. Egal, ob ein HTML-Dokument in einem Browser aufgerufen oder in eine App integriert wird, TalkBack sagt die Überschriften immer auf die gleiche Weise an.
